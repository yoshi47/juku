## 機能と使い方
- ユーザー（生徒・先生）登録・ログイン機能
- 授業予定の登録

### ロールベースのアクセス制御

- **生徒**: 生徒個人のスケジュールを閲覧可能です。
- **先生**: 全生徒・先生のスケジュールを閲覧可能です。
- **塾長**: 先生のすべての機能に加え、授業予定の登録が可能です。
